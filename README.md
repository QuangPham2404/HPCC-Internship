# HPCC-Internship

This script contains benchmarking scripts for the HPCC GAAS cluster to compare the performance of containerized workflows against their non-containerized counterparts. The benchmarks include PyTorch, vLLM, HPC, and HPCG

Hardware (per compute node - kept vague due to possible confidentiality reasons): 2 Intel CPUs (56 cores), 8 H200s NVL, 9 NICs
